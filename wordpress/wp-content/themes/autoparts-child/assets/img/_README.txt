ЛОГОТИП САЙТА
=============

Куда класть:  wp-content/themes/autoparts-child/assets/img/
Формат:       .svg (лучше), можно .png / .webp

Имена файлов (строго так):

  logo.svg          -> логотип в шапке (высота выводится 40px)
  logo-footer.svg   -> логотип в футере (необязательно; если нет — берётся logo.*,
                       к нему применяется серый фильтр автоматически)

Если файлов нет — выводится текстовый логотип «MAJOR SERVICE77».


КАРТИНКА «?» ДЛЯ БАННЕРА «НЕ НАШЛИ НУЖНУЮ ЗАПЧАСТЬ?»
====================================================

Куда класть:  wp-content/themes/autoparts-child/assets/img/
Имя файла:    help-question.png   (можно .webp / .svg)
Формат:       PNG с ПРОЗРАЧНЫМ фоном (зелёный 3D-«?» на чёрном пьедестале)
Размер:       ~260x180 px (в баннере выводится высотой 132px, чуть выступает вверх)

Если файла нет — показывается обычный зелёный «?» (CSS).
