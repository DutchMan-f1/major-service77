ИКОНКИ КНОПОК (шапка и поиск)
=============================

Куда класть:  wp-content/themes/autoparts-child/assets/icons/
Формат:       .svg (инлайнится в разметку)
Цвет:         рисуй линии через stroke="currentColor" (или fill="currentColor").
              Тогда иконка сама подстроится под цвет кнопки:
              тёмная на серых кнопках, белая на корзине/кнопке «Найти».
Размер:       viewBox 0 0 24 24, без фиксированного width/height (масштабируется).

Имена файлов (строго так):

  search.svg     -> лупа (кнопка «Найти» и поиск в шапке)
  mail.svg       -> конверт (соц-кнопка почта)
  telegram.svg   -> telegram (соц-кнопка)
  phone.svg      -> трубка (соц-кнопка + звонок)
  bag.svg        -> корзина/пакет (кнопка корзины)
  user.svg       -> профиль (опционально)

Если файла нет — используется встроенная иконка по умолчанию.
