ИКОНКИ КАТЕГОРИЙ (6 больших карточек на странице модели)
=========================================================

Куда класть:  wp-content/themes/autoparts-child/assets/cats/
Формат:       .svg (лучше) или .png / .webp
Размер:       ~46x46 px (квадрат). Если .svg с currentColor — покрасится сам.

Имена файлов (строго так):

  engine.svg        -> ДВИГАТЕЛЬ
  body.svg          -> КУЗОВ
  chassis.svg       -> ХОДОВАЯ ЧАСТЬ
  electric.svg      -> ЭЛЕКТРИКА
  transmission.svg  -> ТРАНСМИССИЯ
  ac.svg            -> КОНДИЦИОНЕР

Если файла нет — показывается встроенная линейная иконка.
