# Auto Fitment & Baz-on Sync

Плагин авто-магазина запчастей для WordPress + WooCommerce:

- **Каталог авто** — CPT `vehicle` + таксономии `car_make` / `car_model` / `car_generation`.
- **Фитмент** — связь «товар ↔ совместимый автомобиль» (таблица `wp_part_fitment`), фильтрация каталога по выбранному авто.
- **Гараж** — вкладка личного кабинета WooCommerce с сохранёнными авто (таблица `wp_user_garage`).
- **Модалки входа/регистрации** — AJAX, без перезагрузки.
- **Синхронизация baz-on** — WP-Cron каждые 30 минут тянет CSV поставщика, обновляет цены/наличие, создаёт новые товары. Потоковый парсинг + батчи с самоперепланированием (не падает на больших фидах).

## Установка

1. Установить и активировать **WooCommerce** (Плагины → Добавить новый).
2. Активировать плагин **Auto Fitment & Baz-on Sync** — создаст таблицы, эндпоинт «Гараж» и расписание cron.
3. Активировать тему **AutoParts Child** (Внешний вид → Темы). Родитель — Twenty Twenty-Four; можно заменить на Storefront/другую Woo-тему в `style.css` (строка `Template:`).

## Настройка синхронизации

**Каталог авто → Синхронизация baz-on**:

1. Вставить прямую **ссылку на CSV** от baz-on.
2. Задать **разделитель** (обычно `;`), **кодировку** (`auto` — определит; часто Windows-1251).
3. В блоке **Маппинг колонок** указать названия колонок из CSV (или номера с 0, если без заголовков). Обязателен только **Артикул (SKU)**.
4. Сохранить → «Синхронизировать сейчас» для первого прогона.

> ⚠️ Нужен **реальный системный cron**. WP-Cron срабатывает только при заходах на сайт. Для стабильного обновления раз в 30 минут отключите псевдо-cron и повесьте системный:
> ```
> // wp-config.php
> define('DISABLE_WP_CRON', true);
> ```
> Планировщик (каждые 5 мин, WP сам решит когда пора): `wp cron event run --due-now` или прямой вызов `wp-cron.php`.

## Шорткоды

| Шорткод | Назначение |
|---|---|
| `[af_vehicle_selector]` | Каскадный подбор: марка → модель → модификация |
| `[af_auth_buttons]` | Кнопки «Войти / Регистрация» (открывают модалку) |
| `[af_header_tools]` | Подбор + авторизация + мини-корзина (для шапки, из темы) |

## WP-CLI

```
wp af sync            # один проход синхронизации
wp af sync --full     # полный ресинк (скачать фид заново)
wp af sync --loop     # прогоны до завершения
wp af status          # статус последней синхронизации
```
