Логотипы перевозчиков
=====================

Положите сюда официальные файлы логотипов — они автоматически появятся
на оформлении заказа вместо буквенного значка:

    cdek.svg      (или cdek.png / cdek.webp)      — СДЭК
    dellin.svg    (или dellin.png / dellin.webp)  — Деловые Линии

Требования:
- Имя файла = идентификатор перевозчика (cdek, dellin).
- Квадратный или близкий к квадрату логотип, прозрачный фон (для SVG/PNG).
- Показывается в чипе 46×46 px на белой подложке.

Официальные логотипы берите из фирменного гайдлайна перевозчика
(brandbook) — у СДЭК и Деловых Линий есть разделы для партнёров с
готовыми файлами. Использование логотипов допустимо для обозначения
подключённого способа доставки в вашем магазине.
